# AI-UIT

A collection of AI-related problems, experiments, and solutions for learning, practicing, and building a stronger foundation in Artificial Intelligence.

## Overview

**AI-UIT** is a repository that gathers AI problems, exercises, and projects related to studying and practicing Artificial Intelligence at UIT.

The main goal of this repository is to keep everything in one place:
- course-related AI problems
- practice tasks
- experiments
- algorithm implementations
- model training notebooks
- small projects and explorations

This repo serves as both a personal learning archive and a long-term knowledge base for AI topics.

## Objectives

- Collect and organize AI problems in one repository
- Practice core AI concepts through implementation
- Track learning progress over time
- Build reusable code for future projects
- Create a structured reference for studying and revision

## What this repository may contain

Depending on the subject and progress, this repository can include:

- Search algorithms
- Classical AI problems
- Machine learning exercises
- Deep learning experiments
- Data preprocessing
- Model training and evaluation
- Computer vision tasks
- Natural language processing tasks
- Reinforcement learning basics
- Kaggle-style experimentation
- Utility scripts and notebooks
